# Fredrick Kitonyi — Sovereign ICT Architect

Personal portfolio and digital HQ for Fredrick Kitonyi Kiio, Founder & CEO of FRIMAT TECHNOLOGIES.

**Live site:** https://fredrickkitonyi.com

## Tech Stack

- Vite
- React 18 + TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Local Development

```sh
# 1. Clone the repository
git clone <YOUR_GIT_URL>

# 2. Move into the project directory
cd <YOUR_PROJECT_NAME>

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
```

## Build

```sh
npm run build
```

The production output is written to `dist/` and can be deployed to any static host.
